#  Auto Encoder — Pixel Masking Reconstruction

Deep Learning project based on AutoEncoders for image reconstruction and pixel masking using Python and TensorFlow/Keras.

##  Project Overview

This project implements an AutoEncoder neural network capable of reconstructing masked or corrupted image pixels.
The model learns compressed image representations and restores missing information automatically.

##  Features

* Image preprocessing and normalization
* Pixel masking simulation
* Encoder / Decoder architecture
* Image reconstruction using AutoEncoders
* Visualization of original vs reconstructed images
* Deep Learning implementation with TensorFlow/Keras

##  Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Jupyter Notebook

##  Project Structure

* `Auto_Encoder_Masquage_Nourhene_Ayari_IDS_4.ipynb` → Main notebook containing preprocessing, model training, and evaluation

##  Objective

The goal of this project is to explore unsupervised deep learning techniques for image restoration and representation learning using AutoEncoders.
